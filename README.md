## 🏗️ Quick Start

### Para Usuarios
```bash
# Acceder a servicios de IA médica
curl -X POST "https://api.panacea-icono.org/ai/process" \
  -H "Content-Type: application/json" \
  -d '{"text": "análisis médico requerido", "task": "diagnostic"}'
```

### Para Desarrolladores
```bash
# Clonar repositorio principal
git clone https://github.com/panacea-icono/panacea-icono.git
cd panacea-icono

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar API locally
python -m uvicorn main:app --host 0.0.0.0 --port 8000
```

### Para la Comunidad DAO
1. **[Obtener PANAS](./docs/tokenomics/README.md)** - Adquiere tokens para governance
2. **[Unirse al DAO](./docs/governance/README.md)** - Participa en decisiones
3. **[Proponer mejoras](./docs/governance/README.md#tipos-de-propuestas)** - Contribuye al ecosistema
4. **[Stakear tokens](./docs/governance/README.md#participación-en-el-dao)** - Gana recompensas
