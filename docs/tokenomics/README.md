# 🪙 PANAS Token Omics

## Visión General

PANAS (Panacea Algorand Stable Token) es el token nativo del ecosistema Panacea | Icono SA, diseñado para facilitar transacciones, governance y utilidad dentro de nuestro ecosistema de soluciones de salud impulsadas por IA.

## Especificaciones Técnicas

- **Símbolo**: PANAS
- **Blockchain**: Algorand
- **Tipo**: Utility Token / Stable Token
- **Estándar**: ASA (Algorand Standard Asset)
- **Suministro Total**: 1,000,000,000 PANAS

## Casos de Uso

### 1. Governance
- Votación en propuestas del DAO
- Participación en decisiones de protocolo
- Staking para derechos de governance

### 2. Utilidad en Plataforma
- Pagos por servicios de IA médica
- Acceso a herramientas premium
- Descuentos en servicios del ecosistema

### 3. Recompensas
- Incentivos por participación en la comunidad
- Rewards para validadores y contribuyentes
- Bonificaciones por adopción temprana
